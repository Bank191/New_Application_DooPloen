DooPloen Loading Animation

สร้างอนิเมชั่นโหลดที่สวยงามพร้อมเอฟเฟกต์และคำสั่งควบคุมที่ง่ายต่อการใช้งาน

📦 วิธีการใช้งาน

1. เพิ่ม CSS และ JS ลงในไฟล์ HTML ของคุณ

<link rel="stylesheet" href="/css/loading.css"> <script src="/js/loading.js"></script>

▶ แสดง Loader

เรียกคำสั่ง DooPloen.show() เพื่อแสดง loader

DooPloen.show(); // แสดง Loader DooPloen.hide(); // ซ่อน Loader

⏱ แสดง Loader ระยะเวลากำหนด

แสดง Loader แล้วซ่อนอัตโนมัติหลังเวลาที่กำหนด (มิลลิวินาที)

DooPloen.duration(2000); // 2 วินาที DooPloen.duration(5000); // 5 วินาที

🌐 ใช้ Loader กับ API Calls

แสดง Loader เมื่อดึงข้อมูลจาก API

DooPloen.fetch('/api/users') // ดึงข้อมูลและแสดง Loader .then(response => response.json()) .then(data => console.log(data));

⏳ ใช้ Loader กับ Promise/Async

แสดง Loader ขณะรอ Promise ให้เสร็จสิ้น

const promise = new Promise(resolve => { setTimeout(() => resolve(), 3000); }); DooPloen.waitFor(promise);
